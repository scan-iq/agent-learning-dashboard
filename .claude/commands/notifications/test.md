# notifications-test

Test notification channels to verify configuration and connectivity.

## Usage

```bash
/notifications-test [--channel <type>] [--message <text>]
```

## Options

- `--channel` - Specific channel: whatsapp, email, slack, sms, discord, all (default: all)
- `--message` - Custom test message
- `--priority` - Test priority: low, medium, high, critical
- `--verbose` - Show detailed test results

## Examples

### Test all notification channels
```bash
/notifications-test
```

### Test WhatsApp only
```bash
/notifications-test --channel whatsapp
```

### Test with custom message
```bash
/notifications-test \
  --channel email \
  --message "Testing IRIS notifications" \
  --verbose
```

### Test critical priority
```bash
/notifications-test --priority critical
```

## What This Does

1. **Channel Discovery**: Identifies configured channels
2. **Connection Test**: Verifies channel connectivity
3. **Message Send**: Sends test notification
4. **Delivery Verification**: Confirms successful delivery
5. **Performance Measurement**: Measures delivery latency
6. **Error Reporting**: Reports any issues

## When to Use

- **Initial Setup**: Verify configuration after setup
- **Troubleshooting**: Debug notification issues
- **Regular Testing**: Ensure channels remain operational
- **Before Critical Events**: Verify before important notifications
- **After Config Changes**: Test after modifying settings
