# Reflexion Commands

Commands for tracking, searching, and reusing reflexion trajectories for continuous learning.

## Available Commands

- `/reflexion-track` - Track a new reflexion trajectory
- `/reflexion-search` - Search for similar past reflexions
- `/reflexion-stats` - View reflexion system statistics
- `/reflexion-reuse` - Apply reflexion learning to new context
- `/reflexion-analyze` - Analyze trajectory patterns
- `/reflexion-export` - Export reflexions for analysis

## Quick Start

```bash
# Track a new reflexion
/reflexion-track --task "API implementation" --verdict passed

# Search for similar reflexions
/reflexion-search --query "authentication patterns"

# View system stats
/reflexion-stats
```

## Core Concepts

**Reflexion Monitoring** enables:
- Trajectory capture and analysis
- Self-improvement through reflection
- Pattern recognition from experience
- Knowledge reuse across contexts
- Continuous learning from outcomes
