# iris-experts

List and analyze all expert models across projects.

## Usage

```bash
/iris-experts [--filter <criteria>] [--sort <field>] [--export]
```

## Options

- `--filter` - Filter experts: active, healthy, drifting, all (default: active)
- `--sort` - Sort by: performance, accuracy, usage, name (default: performance)
- `--export` - Export format: json, csv, markdown
- `--detailed` - Show detailed expert information

## Examples

### List all active experts
```bash
/iris-experts
```

### Show drifting experts
```bash
/iris-experts --filter drifting --sort accuracy
```

### Detailed expert analysis
```bash
/iris-experts --detailed --export markdown
```

## What This Does

Lists all registered expert models with:
- Performance metrics
- Drift status
- Training history
- Usage statistics
- Specializations
