# Consensus Commands

Commands for managing multi-expert consensus decision tracking and version lineage.

## Available Commands

- `/consensus-record` - Record a new multi-expert consensus decision
- `/consensus-calculate` - Calculate consensus score for decisions
- `/consensus-lineage` - Track expert version lineage and evolution
- `/consensus-rotation` - Get expert rotation recommendations
- `/consensus-search` - Search historical consensus decisions
- `/consensus-quality` - Analyze consensus decision quality

## Quick Start

```bash
# Record a consensus decision
/consensus-record

# View expert lineage
/consensus-lineage iris-prime-console

# Get rotation recommendations
/consensus-rotation
```

## Core Concepts

**Consensus Tracking** enables:
- Multi-expert decision validation
- Version lineage for experts
- Quality scoring for decisions
- Expert rotation optimization
- Historical decision analysis

Works with:
- IRIS for expert coordination
