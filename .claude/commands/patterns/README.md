# Pattern Discovery Commands

Commands for discovering, analyzing, and transferring successful patterns across projects.

## Available Commands

- `/patterns-discover` - Deep pattern discovery from trajectories
- `/patterns-transfer` - Transfer patterns between projects
- `/patterns-learn` - Learn from pattern library

## Quick Start

```bash
# Discover new patterns
/patterns-discover

# Transfer pattern to project
/patterns-transfer --pattern concurrent-ops --target new-project

# Learn from library
/patterns-learn --domain supabase
```

## Core Concepts

**Pattern Discovery** enables:
- Automatic identification of successful approaches
- Cross-project knowledge transfer
- Best practice codification
- Continuous improvement

## Integration

Works with:
- IRIS for expert training
- Reflexion monitoring for source data
- Consensus tracking for validation
- Supabase for persistence
